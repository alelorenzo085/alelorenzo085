# Security Policy

## Supported Versions

This section outlines which versions of the project are currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

You can submit a detailed report through the repository’s Issues section or via the project’s official contact channel (if applicable).

When reporting a vulnerability, please include:

- A clear description of the issue
- Steps to reproduce the problem
- The potential impact
- A suggested fix (if available)

## What to Expect After Reporting

- Acknowledgment of receipt within a reasonable timeframe
- Evaluation of the reported issue
- Notification whether the vulnerability is accepted or declined
- If accepted, a fix will be developed and released in a supported version
